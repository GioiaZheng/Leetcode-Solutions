#  LeetCode Solutions 
This repository contains my personal LeetCode solutions written in Python 3.

##  Structure
Each problem has its own folder containing:
- `README.md` — Full explanation, examples, diagrams
- `solution.py` — Clean and efficient Python implementation

##  Topics Covered
- Dynamic Programming (DP)
- Graphs & Trees
- Greedy Algorithms
- Backtracking
- Prefix/Suffix techniques
- Sliding Window
- Binary Search
- Hashing
- Bitmask
- Math & Combinatorics

##  Goal
Build a complete, well-documented algorithm notebook to master technical interviews.
