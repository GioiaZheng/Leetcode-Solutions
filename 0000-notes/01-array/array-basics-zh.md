# 数组基础（中文版）

数组题核心是：**顺序是否重要、是否连续、是否可原地修改**。

## 核心概念

- 数组是有序容器，索引天然表示顺序。
- 大多数题可从这三问切入：
  1. 要不要保持原顺序？
  2. 要不要连续区间？
  3. 需不需要额外空间？

## 常见解法映射

- 一次遍历：维护最值、计数、前缀状态。
- 双指针：有序数组或区间收缩。
- 滑动窗口：连续子数组统计/最优化。
- 前缀和：快速求区间和。

## 示例 1：一次遍历求最大利润

```python
min_price = float('inf')
ans = 0
for x in prices:
    min_price = min(min_price, x)
    ans = max(ans, x - min_price)
```

思路：到当前位置 `x`，只关心“之前最低买入价”。

## 示例 2：原地删除元素

```python
k = 0
for x in nums:
    if x != val:
        nums[k] = x
        k += 1
# 前 k 个即答案
```

## 易错点

- 把子数组与子序列混淆。
- 更新顺序写反（先更新答案还是先更新状态）。
- 忘了空数组和全相同元素。

## 练习建议

先按类别刷：
1. 一次遍历维护状态
2. 双指针原地覆盖
3. 前缀和计数问题
