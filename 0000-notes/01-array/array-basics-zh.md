# 数组基础（中文版）

本笔记聚焦数组题的思维方式与常见套路。英文详版见：[array-basics.md](array-basics.md)。

---

## 核心概念

- 数组表示**有序序列**，顺序是主要信息。
- 先判断：**是否必须利用顺序**？
  - 需要 → 一次遍历 / 双指针 / 滑窗 / 前缀和
  - 不需要 → 排序 / 哈希 / 计数

---

## 关键思维

- 始终问自己：**“到 i 为止我知道了什么？”**
- 常维护信息：前缀和、当前最值、计数、频次等。

---

## 常见模板

```python
for i in range(n):
    # 处理 nums[i]
```

- 一次遍历能解决很多问题。
- 注意索引边界与空数组情况。

---

## 易错点

- i/j 指针移动顺序错误
- 题目要求“子数组”却写成“子序列”
- 忘记考虑负数/重复元素
