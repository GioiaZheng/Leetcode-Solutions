# 前缀和（中文版）

前缀和用于快速计算区间和，把多次 O(k) 查询降到 O(1)。

## 定义

`pre[i]` 表示前 `i` 个元素和（`pre[0]=0`）。

则区间 `[l, r]` 和为：
`pre[r+1] - pre[l]`。

## 模板

```python
pre = [0]
for x in nums:
    pre.append(pre[-1] + x)
```

## 例子：和为 k 的子数组个数

- 维护当前前缀和 `s`。
- 需要找之前是否出现过 `s-k`。
- 哈希统计前缀和频次。

## 易错点

- 下标偏移（`r+1` 很关键）。
- 忘记初始化 `cnt[0]=1`。
- 负数场景下不能用双指针替代。
