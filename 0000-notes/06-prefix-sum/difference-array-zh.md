# 差分数组（中文版）

差分适合“多次区间加减、最后统一还原”的场景。

## 定义

原数组 `a`，差分 `d`：
- `d[0] = a[0]`
- `d[i] = a[i] - a[i-1]`

若区间 `[l, r]` 加 `x`：
- `d[l] += x`
- `d[r+1] -= x`（若存在）

最后对 `d` 做前缀和即可还原 `a`。

## 例子

对长度 5 的全 0 数组执行：
1. `[1,3] +2`
2. `[2,4] +1`

最终数组为 `[0,2,3,3,1]`。

## 易错点

- `r+1` 越界判断。
- 下标从 0 还是 1 开始。
- 忘记最后还原步骤。
