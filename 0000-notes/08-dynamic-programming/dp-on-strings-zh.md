# 字符串 DP（中文版）

字符串 DP 常用二维状态，按区间长度或前缀长度推进。

## 常见问题

- 最长公共子序列（LCS）
- 编辑距离
- 回文子串/回文子序列

## 例子：LCS

`dp[i][j]`：`s[:i]` 与 `t[:j]` 的 LCS 长度。

转移：
- 若 `s[i-1]==t[j-1]`：`dp[i][j]=dp[i-1][j-1]+1`
- 否则：`max(dp[i-1][j], dp[i][j-1])`

## 遍历顺序

一般 `i` 从 1..n，`j` 从 1..m。

## 易错点

- 字符串下标与 dp 下标错位。
- 初始边界（第 0 行/列）未设为 0。
