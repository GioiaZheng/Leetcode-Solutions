# 动态规划总结（中文版）

## 什么时候用 DP

- 有阶段性决策。
- 子问题重复出现。
- 能写出“当前状态由过去状态转移”。

## 快速流程

1. 写状态定义（最重要）。
2. 写转移方程。
3. 定义 base case。
4. 按依赖顺序迭代。
5. 必要时做空间优化。

## 常见类型速查

- 一维线性 DP：爬楼梯、打家劫舍。
- 二维网格 DP：最短路径和。
- 区间 DP：回文、矩阵链。
- 背包 DP：容量约束选择。

## 实战建议

- 先写记忆化搜索验证思路，再改迭代。
- 对每道题都写一句“状态语义”。
## 小例子

“最小路径和”就是典型网格 DP：
- `dp[i][j]` 表示到 `(i,j)` 的最小和。
- 转移来自上方和左方最小值。
