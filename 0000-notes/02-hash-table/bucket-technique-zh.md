# 桶思想（中文版）

桶是“按范围分组”技巧，常用于近似比较、线性时间去重/查邻近值。

## 核心思想

把值映射到桶编号：
- 同桶元素差值一定较小。
- 只需检查本桶和相邻桶。

## 典型题

Contains Duplicate III（值差 ≤ t 且下标差 ≤ k）。

## 桶编号示意

令桶宽 `w = t + 1`：
- `id = floor(x / w)`（负数需小心处理）
- 若同桶已有元素，必满足值差 ≤ t。

## 为什么快

每个元素最多比较 3 个桶，整体 O(n)。

## 易错点

- `t = 0` 时桶宽为 1。
- 滑动窗口超出 k 时，要删除旧元素所在桶。
- 负数整除在不同语言行为不同，要统一实现。
## 小例子

数组 `[1,5,9,1]`，`k=3,t=3`：
- 桶宽 `w=4`。
- 当处理到最后一个 `1` 时，可在窗口内发现同桶/邻桶的旧值并判断是否满足条件。
