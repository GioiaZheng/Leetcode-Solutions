# 区间贪心（中文版）

区间问题中最经典策略：**按右端点排序**。

## 为什么按右端点

右端点越小，给后续区间留下的空间越大。

## 模板（最多不重叠区间）

```python
intervals.sort(key=lambda x: x[1])
ans = 0
end = -10**18
for l, r in intervals:
    if l >= end:
        ans += 1
        end = r
```

## 例子：删除最少区间使其不重叠

- 先算最多可保留的不重叠区间数 `keep`。
- 答案为 `n - keep`。

## 易错点

- 闭区间/开区间边界判断不同（`>=` 或 `>`）。
- 排序后忘记更新当前右端点。
