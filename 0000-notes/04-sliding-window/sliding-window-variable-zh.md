# 可变长度滑动窗口（中文版）

当题目是“满足条件的最短/最长连续区间”时，常用可变窗口。

## 核心套路

- 右指针扩张，直到满足（或不满足）条件。
- 左指针收缩，维持窗口合法。
- 每一步更新答案。

## 通用模板

```python
l = 0
for r, x in enumerate(nums):
    # 加入 nums[r]
    while 条件不合法:
        # 移除 nums[l]
        l += 1
    # 更新答案
```

## 例子：最长无重复子串

- 用哈希记录字符频次。
- 若某字符频次 > 1，不断移动 `l` 直到恢复合法。

## 易错点

- “何时更新答案”位置放错。
- 条件写成 `if` 而不是 `while`。
- 忘记处理频次减到 0 的键。
